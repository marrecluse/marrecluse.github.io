# Linux Admin Toolbox

A collection of Bash scripts to automate system administration tasks on Linux systems.

## Features

- Display system info
- Manage users
- Monitor services
- Back up and clean logs
- Auto-update packages
- Audit network and firewall

## Usage

```bash
chmod +x <script_name>.sh
./<script_name>.sh
```

## Setup (Optional Cron Jobs)

```
0 2 * * * /path/to/backup_tool.sh
```
